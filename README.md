# Frutilla Millonaria (Frontend Gráfico)

Proyecto frontend construido con React + Vite + Phaser.

## Requisitos

- Node.js 20 o superior
- npm 10 o superior

## Ejecutar en local

1. Abrir una terminal en la carpeta del proyecto:
   `FrutillaMillonaria-Grafica`
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Levantar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abrir en navegador la URL que muestra Vite (normalmente):
   `http://localhost:5173`

## Scripts útiles

- `npm run dev`: inicia el proyecto en modo desarrollo.
- `npm run build`: genera build de producción en `dist/`.
- `npm run preview`: previsualiza el build de producción.
- `npm run lint`: ejecuta ESLint.
